
# LenssHub

**LenssHub** is an online eyewear e-commerce website built as a team project. It provides users with a clean, intuitive interface to browse and explore a wide range of optical products, such as spectacles, glasses, frames, lenses, and sunglasses. The platform has a responsive UI and uses a combination of front-end and back-end technologies to deliver a complete shopping experience.
